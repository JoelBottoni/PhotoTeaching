# PhotoTeaching

This repository contains teaching materials designed for education and training in forensic photography. Available in English and German.

## Content

- e-learning_photography_EN.pptx - Narrated teaching powerpoint about forensic photography
- e-learning_Fotografie_DE.pptx - Deutsche Version der PowerPoint-Präsentation
